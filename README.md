# JAVASCRIPT-PROJECTS
This repository is some beginner projects I made for my Tech Academy JavaScript course.

## Projects

### [Pizza Menu Project](Pizza_Project/)
An interactive pizza ordering menu application built with JavaScript, HTML, and CSS. This project demonstrates form handling and dynamic menu selection features.

### [Tic Tac Toe Game](Tic-Tac-Toe/)
A classic Tic Tac Toe game implementation featuring game logic, win detection, and player turn management.

### [Calculator Project](Calculator_Project/)
A functional calculator application for performing basic arithmetic operations including addition, subtraction, multiplication, and division.
